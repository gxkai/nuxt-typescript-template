# nuxt-edge + Typescript Starter

This is a project template for [vue-cli](https://github.com/vuejs/vue-cli).

```sh
$ vue init hmsk/nuxt-typescript-edge-template my-edgey-project
$ cd my-edgey-project
$ npm install # Or yarn
```

- Requires `@vue-cli/init` or `vue-cli@^2.1.0`
